# Modern Developer Portfolio

A stunning, modern single-page portfolio website built with React, Vite, Tailwind CSS v4, and Framer Motion. Features a sophisticated midnight blue & silver theme with glassmorphism, 3D effects, and smooth animations.

- **Modern Design**: Midnight Blue & Silver professional theme
- **Smooth Animations**: Framer Motion animations throughout
- **3D Effects**: React Parallax Tilt for interactive cards
- **Glassmorphism**: Beautiful glass-effect components
- **Responsive**: Fully responsive design for all devices
- **Smooth Scrolling**: Lenis smooth scroll implementation
- **Single Page Application**: All sections on one page with smooth scrolling
